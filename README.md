# Payroll Management System

A modern payroll management platform designed to simplify employee salary processing, payroll tracking, and HR operations.

## Features

- Employee Payroll Management
- Salary Calculation
- Attendance & Work Tracking
- Payslip Generation
- Secure Authentication
- Responsive Dashboard
- Real-time Data Management

---

## Tech Stack

- React.js
- Node.js
- Tailwind CSS
- Firebase / API Integration
